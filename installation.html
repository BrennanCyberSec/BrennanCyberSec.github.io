Kali Purple Installation Guide

Building a purple teaming environment for security research and testing

About This Guide This guide walks you through setting up Kali Purple in a VMware environment, perfect for building your own purple teaming lab for security testing and development.

Prerequisites

Before you begin, ensure your system meets these requirements:

  • Virtualization enabled in BIOS/UEFI
  • 16GB RAM minimum (32GB recommended for multiple VMs)
  • 100GB+ available disk space
  • VMware Workstation or VMware Player installed
  • Kali Purple ISO or pre-built VMware image

Installation Steps

1

Download Kali Purple

Download the latest Kali Purple image from the official Kali Linux website. You can choose between:

  • ISO Image - For fresh installations
  • VMware Image - Pre-configured and ready to go (recommended)

Visit: https://www.kali.org/get-kali/

2

Extract the VMware Image

If you downloaded the VMware image, you'll need to extract it first.

On Linux:

sudo apt update && sudo apt install -y p7zip-full
7z x kali-linux-*-vmware-amd64.7z

On Windows:

  • Download and install 7-Zip
  • Right-click the downloaded file
  • Click "Show more options" (Windows 11)
  • Select "7-Zip" ‚Üí "Extract Here"
3

Import to VMware

Launch VMware and import the extracted virtual machine:

  1. Open VMware Workstation/Player
  2. Click File ‚Üí Open
  3. Navigate to the extracted folder
  4. Select the .vmx file
  5. Click Open
4

Configure VM Settings

Before starting the VM, optimize the settings:

  1. Right-click the VM and select Settings
  2. Increase RAM to at least 4GB (8GB recommended)
  3. Allocate 2-4 CPU cores
  4. Set network adapter to NAT or Bridged
  5. Click OK to save
5

First Boot and Update

Power on the VM and perform initial updates:

sudo apt update && sudo apt full-upgrade -y

This ensures all security tools and the system are up to date.

Security Note Default credentials for Kali Purple: kali:kali
Change this immediately after first login for security.

Post-Installation Setup

Change Default Password

passwd

Install VMware Tools (if needed)

sudo apt install -y open-vm-tools open-vm-tools-desktop
sudo reboot

Configure Purple Team Tools

Kali Purple comes with pre-installed defensive security tools. Access them through:

  • TheHive - Case management platform
  • Elasticsearch - Search and analytics engine
  • Kibana - Data visualization dashboard
  • Suricata - Network IDS/IPS
  • Malcolm - Network traffic analysis
Pro Tip Snapshot your VM after completing the initial setup. This allows you to quickly revert to a clean state when testing different configurations or after malware analysis.
