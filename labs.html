Security Labs

Hands-on exercises for purple teaming, malware analysis, and detection engineering

Purple Team Labs

Active Directory Attack & Defense

Purple Team

Build a complete AD environment with domain controller and workstations. Perform attacks like Kerberoasting, AS-REP Roasting, and Pass-the-Hash, then detect them using Elastic Security and custom detection rules.

2-3 hours Intermediate

Command & Control Detection

Purple Team

Deploy popular C2 frameworks like Sliver and Covenant. Analyze their traffic patterns and build detection rules to identify C2 beaconing in your SIEM.

2-3 hours Advanced

Phishing Campaign Analysis

Purple Team

Create and execute phishing campaigns using GoPhish, then analyze detection capabilities with email security tools and endpoint protection.

1-2 hours Intermediate

Red Team Labs

Malware Analysis Pipeline

Red Team

Set up an isolated malware analysis environment with Remnux and Cuckoo Sandbox. Analyze real malware samples and understand their behavior, IOCs, and TTPs.

1-2 hours Advanced

Custom Payload Development

Red Team

Develop custom malware in C/C++ with techniques like process injection, API hooking, and AV evasion. Test against modern endpoint security solutions.

3-4 hours Advanced

Network Pivoting & Lateral Movement

Red Team

Practice advanced penetration testing techniques including SSH tunneling, port forwarding, and moving laterally through a multi-tier network.

2-3 hours Advanced

Blue Team Labs

SIEM Detection Engineering

Blue Team

Create custom detection rules in Elastic Security for common attack techniques mapped to MITRE ATT&CK. Learn sigma rules and detection-as-code practices.

1-2 hours Intermediate

Network Traffic Analysis

Blue Team

Analyze malicious network traffic using Wireshark, Zeek, and Suricata. Identify suspicious patterns, C2 communications, and data exfiltration attempts.

2 hours Intermediate

Incident Response Workflow

Blue Team

Practice complete incident response procedures using TheHive for case management and Velociraptor for endpoint forensics and threat hunting.

2-3 hours Advanced
